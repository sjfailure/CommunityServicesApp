# KC Connect - Android Client

KC Connect is a native Android application designed to bridge the gap between Kansas City residents and essential community services. Inspired by the Kansas City Public Library "Street Sheets," this app digitizes quarterly service data to provide a searchable, intuitive interface for those seeking assistance.
